# 技能分享 - product-xiaopi

## 技能名称
**n8n 工作流自动化平台**

## 技能特点
1. **开源可自托管** — n8n 是开源的工作流自动化工具，支持 Docker 一键部署，数据完全在自己服务器上，隐私有保障，适合企业级使用
2. **可视化 + 代码双模式** — 提供直观的拖拽式工作流编辑器，同时也支持编写 JavaScript/Python 代码片段，兼顾易用性和灵活性
3. **丰富的集成节点** — 内置 400+ 预构建连接器，覆盖 Google、Slack、GitHub、Notion、飞书、Webhook、HTTP 请求、数据库等常用服务
4. **事件驱动架构** — 支持 Webhook、定时任务、手动触发等多种启动方式，可实现实时响应外部事件触发自动化流程
5. **AI 原生支持** — 内置 AI Agent 节点、向量数据库支持、LangChain 集成，可构建复杂的 AI 工作流（如 RAG、聊天机器人）
6. **错误处理与重试** — 支持可配置的重试策略、错误分支处理、死信队列，确保复杂流程的可靠性

## 应用场景
- **跨平台数据同步**：当 Notion 更新时自动同步到飞书文档、发送 Slack 通知、更新数据库记录
- **审批流程自动化**：员工提交申请 → 触发审批工作流 → 通知审批人 → 记录结果 → 更新系统状态
- **监控告警**：服务器指标异常时自动抓取数据、生成告警卡片、触发飞书/钉钉通知
- **AI 数据处理管道**：定时抓取内容 → 调用 LLM 摘要 → 存入向量数据库 → 提供问答服务
- **定时报表生成**：每天自动从数据库拉取数据 → 生成 Excel/CSV → 发送邮件给相关人
