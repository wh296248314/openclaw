# 技能分享 - deployment-xiaopi

## 技能名称
n8n 工作流自动化平台

## 技能特点
1. **开源可自托管**：支持完全开源自部署，数据完全在自己掌控中，隐私有保障
2. **可视化工作流编辑器**：通过拖拽节点设计自动化流程，无需编写大量代码
3. **丰富的集成能力**：支持 400+ 应用集成，包括 GitHub、Slack、钉钉、飞书、数据库等
4. **代码执行能力**：内置 JavaScript/TypeScript 代码节点，支持复杂业务逻辑
5. **多种触发方式**：支持 Webhook、定时任务、事件触发、API 调用等多种触发方式
6. **错误处理与重试**：内置重试机制、死信队列、错误处理节点，保证任务可靠性

## 应用场景
- **CI/CD 流程编排**：自动化构建、测试、部署流程
- **定时任务调度**：定期数据同步、备份、报表生成
- **跨系统数据同步**：不同系统间的数据流转和格式转换
- **消息通知自动化**：多渠道（钉钉/飞书/邮件/Slack）通知推送
- **Webhook API 开发**：快速构建 API 接口接收外部事件
- **运维自动化**：服务器监控告警、自动化运维任务
