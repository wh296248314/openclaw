### 监控卫士功能
```python
# 在Python代码中使用
from timeout_guardian import TimeoutGuardian

# 创建监控实例
guardian = TimeoutGuardian()

# 开始监控任务
task_id = guardian.start_task(
    task_id="unique_task_id",
    task_name="任务名称",
    description="任务描述"
)

# 更新任务状态
guardian.update_task(
    task_id=task_id,
    status="processing",
    estimated_completion="预计完成时间"
)

# 完成任务
guardian.complete_task(
    task_id=task_id,
    result="任务结果"
)
```
