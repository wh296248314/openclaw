# 虎码输入法手动安装指南

## 安装步骤

### 1. 获取tiger.db文件
由于直接下载链接返回的是HTML页面，需要手动获取文件：

**方法A：从其他来源下载**
1. 访问 http://huma.ysepan.com/
2. 在页面上找到tiger.db文件的下载链接
3. 下载8.5MB的tiger.db文件

**方法B：使用其他下载方式**
```bash
# 尝试使用curl带特定参数
curl -L -o tiger.db "http://huma.ysepan.com/下载链接"

# 或者使用wget带referer
wget --referer="http://huma.ysepan.com/" -O tiger.db "实际文件URL"
```

### 2. 安装文件到系统
```bash
# 复制文件到ibus-table目录
sudo cp tiger.db /usr/share/ibus-table/tables/

# 设置正确的权限
sudo chmod 644 /usr/share/ibus-table/tables/tiger.db
```

### 3. 重启ibus服务
```bash
# 重启ibus守护进程
ibus restart

# 或者重启ibus-daemon
pkill ibus-daemon && ibus-daemon -drx
```

### 4. 添加虎码输入法
1. 打开系统设置 → 区域与语言 → 输入源
2. 点击"+"添加输入源
3. 选择"中文" → "虎码"或"Tiger"
4. 或者使用命令：`ibus-setup`

### 5. 设为默认输入法
```bash
# 设置ibus为默认输入法框架
im-config -n ibus

# 或者在ibus设置中设置默认输入法
```

## 验证安装

### 检查文件
```bash
# 检查文件是否已安装
ls -lh /usr/share/ibus-table/tables/tiger.db

# 检查文件类型
file /usr/share/ibus-table/tables/tiger.db
```

### 检查ibus配置
```bash
# 查看ibus已安装的表格
ibus read-config | grep -i table

# 查看ibus进程状态
ps aux | grep ibus
```

## 故障排除

### 问题1：找不到tiger.db文件
**解决方案**：
1. 确认文件已正确下载（大小应为8.5MB）
2. 确认文件复制到了正确目录
3. 检查文件权限：`ls -l /usr/share/ibus-table/tables/tiger.db`

### 问题2：ibus重启后不显示虎码
**解决方案**：
1. 完全重启ibus：`pkill -9 ibus-daemon && ibus-daemon -drx`
2. 重新登录系统
3. 检查ibus-table是否已安装：`dpkg -l | grep ibus-table`

### 问题3：无法添加输入源
**解决方案**：
1. 运行`ibus-setup`手动配置
2. 在图形界面中添加输入法
3. 检查~/.config/ibus/bus/目录

## 替代方案

如果无法获取tiger.db文件，可以考虑：

### 1. 安装其他中文输入法
```bash
# 安装拼音输入法
sudo apt install ibus-pinyin

# 安装智能拼音
sudo apt install ibus-libpinyin
```

### 2. 使用fcitx输入法框架
```bash
# 安装fcitx
sudo apt install fcitx fcitx-pinyin

# 设置fcitx为默认
im-config -n fcitx
```

## 注意事项

1. **文件来源**：确保从可信来源下载tiger.db文件
2. **文件大小**：正确的tiger.db文件应为8.5MB左右
3. **系统兼容**：确保ibus-table版本与系统兼容
4. **用户权限**：可能需要重启系统或重新登录才能生效
