## 🚀 使用虎码输入法

### 方法一：图形界面添加（推荐）
1. **打开系统设置**
   ```bash
   gnome-control-center region
   # 或点击右上角系统菜单 → 设置
   ```

2. **添加输入源**
   - 进入"区域与语言" → "输入源"
   - 点击"+"按钮
   - 选择"中文" → 查找"虎码"或"Tiger"
   - 点击"添加"

3. **切换输入法**
   - 使用快捷键 `Super + Space` 切换输入法
   - 或点击任务栏输入法图标选择

### 方法二：命令行配置
```bash
# 启动ibus设置界面
ibus-setup

# 在图形界面中添加虎码输入法
```

### 方法三：直接编辑配置
```bash
# 查看当前输入法配置
gsettings get org.gnome.desktop.input-sources sources

# 添加虎码输入法（如果需要）
# 注意：这需要知道虎码输入法的具体ID
```

## ⚙️ 配置优化

### 设置默认输入法
```bash
# 如果ibus不是默认，设置为默认
im-config -n ibus

# 重启ibus确保生效
pkill ibus-daemon && ibus-daemon -drx &
```

### 快捷键配置
1. 打开 `ibus-setup`
2. 进入"高级"选项卡
3. 配置切换快捷键：
   - 下一个输入法: `Super+Space`
   - 上一个输入法: `Shift+Super+Space`

### 输入法选项
在ibus设置中可配置：
- 候选词数量
- 联想功能
- 模糊音设置
- 皮肤主题

## 🐛 故障排除

### 问题1：找不到虎码输入法
**解决方案**：
```bash

pkill -9 ibus-daemon
ibus-daemon -drx &

# 2. 重新登录系统
# 3. 检查文件权限
ls -l /usr/share/ibus-table/tables/tiger.db
```

### 问题2：输入法不显示候选词
**解决方案**：
1. 检查ibus-table是否安装完整
2. 尝试其他应用程序测试
3. 查看错误日志：`tail -f ~/.cache/ibus/ibus-daemon.log`

### 问题3：快捷键冲突
**解决方案**：
1. 在 `ibus-setup` 中修改快捷键
2. 检查系统快捷键设置
3. 避免与其他软件快捷键冲突
