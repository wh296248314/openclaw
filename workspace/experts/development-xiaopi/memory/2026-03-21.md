## 军机处 (Sansheng Demo) 维护

### Docker部署问题修复
- **问题**: Docker镜像 `cft0808/sansheng-demo:latest` 是 ARM64 架构，与 AMD64 主机不兼容
- **解决**: 使用源码构建本地AMD64镜像
  ```bash
  cd /home/pixiu/edict
  sudo docker build --platform linux/amd64 -t sansheng-demo:local .
  ```

### 数据卷挂载问题
- **问题**: Docker容器没有挂载数据目录，使用内置演示数据
- **解决**: 更新docker-compose.yml，添加数据卷挂载
  ```yaml
  volumes:
    - ./data:/app/data
  ```

### 权限问题
- **问题**: 容器内appuser无法读取宿主数据文件
- **解决**: `sudo chmod -R 777 /home/pixiu/edict/data/`
