# 教育行业SCRM竞品分析

> 更新时间：2026-03-23  
> 参考：竞品小程序功能调研报告（2025年6月）+ 尘锋/微盛/探马公开资料

## 一、执行摘要

### 核心结论

**市面上没有专门针对 K12+全日制教育机构的 SCRM 产品，存在明显市场空白。**

| 竞品 | 定位 | 教育适配度 | 价格 |
|------|------|------------|------|
| 尘锋SCRM | 中大型企业 | ★★★☆☆ | 3-50万/年 |
| 微盛SCRM | 中小企业 | ★★☆☆☆ | 3千-3万/年 |
| 探马SCRM | B2B销售团队 | ★☆☆☆☆ | 2-10万/年 |

### 机会点

> **营销 + 销售 + 服务 + 教务一体化** 是教育行业真正需要的。

## 三、教育行业专项对比

### 3.1 报读流程对比

| 竞品 | 流程特点 | 评价 |
|------|----------|------|
| **高途** | 直接结算，最简洁 | ⭐ 最优 |
| **学而思** | 需诊断后报名（A班可直接） | ⭐ 合理 |
| **蓝天** | 需预约线下诊断 | 一般 |
| **新东方** | 只能咨询，无法直接报读 | ❌ 差 |

> **自建建议**：支持直接报读 + 保留预约诊断入口

### 3.2 优惠策略对比

| 策略 | 代表竞品 | 说明 |
|------|----------|------|
| **老带新** | 学而思 | 面对面邀请 + 海报分享邀请 |
| **拼团** | 蓝天 | 拼团砍价为主 |
| **顾问联动** | 高途/学而思 | 加购后立即分配顾问跟进 | ← 关键

> **自建建议**：老带新 + 拼团 + 顾问联动

### 3.3 家校服务对比

| 功能 | 代表竞品 | 评价 |
|------|----------|------|
| **成长档案** | 学而思/高途 | 有课后反馈和成长记录 ⭐ |
| **校区查询** | 蓝天 | 直接引流顾问企微 ⭐ 最优 |
| **调课退费** | 新东方/学而思 | 支持线上操作 |

> **自建建议**：课后反馈 + 校区查询 + 顾问联动 + 退费申请

## 附录：核心业务流程

### 获客转化流程
```
广告投放 → 落地页 → 扫码
    ↓
活码承接（自动打标签 + 发送欢迎语）
    ↓
销售SOP（1天发资料 → 3天发案例 → 7天发优惠 → 14天预约体验）
    ↓
体验转化（预约 → 上课 → 测评报告 → 报价 → 签单）
    ↓
服务运营（入学 → 课后反馈 → 续费提醒 → 续费）
```

### 家校服务流程
```
入学 → 建立档案 → 入群
    ↓
日常服务（课后反馈 + 成长档案 + 节日关怀）
    ↓
续费节点（提前30天提醒 → 专属优惠 → 1v1沟通）
    ↓
转介绍激发（老带新 + 分享奖励）
```
