# 标准workspace目录模板

## 完整结构

```
workspace/
├── # 核心配置文件
├── AGENTS.md              # 机器人配置
├── SOUL.md                # 身份定义
├── USER.md                # 用户信息
├── TOOLS.md               # 工具配置
├── MEMORY.md              # 长期记忆
├── HEARTBEAT.md           # 心跳任务
├── BOOTSTRAP.md           # 初始化（用完可删）
│
├── # 专家小皮目录
├── experts/
│   ├── product-xiaopi/    # 产品小皮
│   ├── design-xiaopi/    # 设计小皮
│   ├── development-xiaopi/ # 研发小皮
│   ├── testing-xiaopi/   # 测试小皮
│   ├── deployment-xiaopi/ # 部署小皮
│   └── audit-xiaopi/     # 审核小皮
│
├── # 技能目录
├── skills/
│   └── {skill-name}/
│       ├── SKILL.md
│       ├── scripts/
│       ├── references/
│       └── assets/
│
├── # 共享资源
├── shared/
│   ├── notes/             # 团队规范文档
│   ├── templates/        # 模板
│   ├── assets/           # 静态资源
│   ├── data/             # 数据文件
│   └── archive/         # 归档文件
│
├── # 知识库
├── knowledge/
│
└── # 日常记忆
└── memory/
    └── YYYY-MM-DD.md
```

## 各小皮目录标准

```
{xiaopi}/
├── AGENTS.md          # 必须
├── SOUL.md            # 必须
├── USER.md            # 必须
├── MEMORY.md          # 必须
├── HEARTBEAT.md       # 推荐
├── TOOLS.md           # 可选
├── knowledge/         # 可选
└── workspace/        # 可选
```
