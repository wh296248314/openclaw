## 补充版团队工作规范

### 1. 任务接收规范
- 收到任务 → 回复"收到，预计X分钟"
- 超过1分钟 → 主动汇报进度

### 2. 汇报格式规范
- 结论先行：先说结论，再给背景
- 格式：结论 → 皮休决策
- 禁止：长篇大论让皮休自己总结

### 3. 重要事项规范
- 发 Interactive Card → 同步记录到 memory
- 关键信息 → 立即记，不等问
- 发送内容 → 提醒皮休是否需要记录

### 4. Interactive Card 记录规范
- 发送时间
- 发送目的
- 卡片内容摘要
- 存放位置：shared/notes/cards/
