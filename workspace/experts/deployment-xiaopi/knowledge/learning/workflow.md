# 学习工作流

## 执行流程

### 1. 定时触发
- 使用 cron 定时任务触发
- 或 heartbeat 中定期执行

### 2. 技能搜索
```
clawhub search <领域关键词>
clawhub explore --limit 15
```

### 3. 筛选评估
- 检查技能描述和功能
- 评估与当前工作的相关性
- 记录有价值的技能

### 4. 发现报告
- 整理发现的技能列表
- 生成简要评估
- 写入 discovered.md

### 5. 用户通知
- 告知皮休发现的技能
- 提供安装建议
- 等待确认后安装
