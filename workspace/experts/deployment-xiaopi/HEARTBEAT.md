### 📋 同步触发条件（必须立即执行）

| 触发场景 | 说明 | 同步时机 |
|---------|------|---------|
| **reset前** | 会话即将重置 | 必须先同步到memory |
| **任务完成** | 收到任务并完成 | 立即同步 |
| **配置变更** | openclaw.json、skills等 | 立即同步 |
| **重要决策** | 涉及部署方案、优先级判断 | 立即同步 |
| **协作结论** | 与其他小皮达成的重要结论 | 立即同步 |

### 📝 同步格式规范

同步到 `memory/YYYY-MM-DD.md`，使用以下格式：

```markdown
## ⏰ [HH:MM] 同步记录

### 触发类型
[reset前/任务完成/配置变更/重要决策/协作结论]

### 同步内容
- 任务/事项：[具体描述]
- 结论/结果：[具体结论]
- 后续行动：[需要跟进的内容]

### 关联会话
[会话关键词/群组]
```

### ✅ 同步检查清单

每次心跳时额外检查：
- [ ] 上次reset到现在是否有重要结论未同步？
- [ ] 是否有任务已完成但未同步？
- [ ] 是否有配置变更但未记录？

### 🔒 同步原则

1. **立即性**：触发条件满足时立即执行，不等待
2. **完整性**：记录关键信息，不要遗漏重要细节
3. **可追溯**：记录来源和上下文，方便后续查阅
4. **可执行**：结论包含明确的行动项
