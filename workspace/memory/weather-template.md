# 天气问候模板

## 基础结构
```
🌅 早安晗哥！皮休已上线 🐾

🌤️ **今日天气**：[天气状况]
🌡️ **温度**：[最低]°C ~ [最高]°C
💨 **风力**：[风力等级]
💧 **湿度**：[湿度]%
🎯 **空气质量**：[AQI指数]

🌟 **个性化祝福**：
[根据天气和日期生成的祝福语]

📋 **今日最重要事项**：
1. [优先级1任务]
2. [优先级2任务] 
3. [优先级3任务]

💡 **温馨提示**：
[根据天气的实用建议]

新的一天，加油！✨
```

## 天气状况对应祝福语

### 晴天 ☀️
- "阳光明媚，心情也会跟着灿烂起来！"
- "好天气带来好心情，今天一定顺利！"
- "阳光正好，适合高效工作的一天！"

### 多云 ⛅
- "云淡风轻，正是专注工作的好时机！"
- "天气温和，不冷不热，工作效率加倍！"
- "多云天气，心情平和，适合深度思考！"

### 雨天 🌧️
- "雨声淅沥，是专注工作的白噪音！"
- "雨天记得带伞，也要记得保持好心情！"
- "雨水滋润万物，也滋润你的创造力！"

### 雪天 ❄️
- "雪花飘飘，世界变得纯净美好！"
- "雪天路滑，出行注意安全！"
- "银装素裹，适合冷静思考的一天！"

### 雾天 🌫️
- "雾气朦胧，但你的目标清晰可见！"
- "雾天行车注意安全，工作也要稳步前进！"
- "迷雾终将散去，就像问题终将解决！"

## 温度对应建议

### 炎热 (>30°C)
- "天气炎热，记得多喝水，注意防暑降温！"
- "高温天气，午间适当休息，保持精力！"

### 温暖 (20-30°C)
- "舒适的温度，适合高效工作！"
- "天气宜人，可以适当开窗通风！"

### 凉爽 (10-20°C)
- "天气凉爽，头脑清醒，适合深度工作！"
- "早晚温差大，记得添衣保暖！"

### 寒冷 (<10°C)
- "天气寒冷，注意保暖，别感冒了！"
- "冷天适合喝杯热茶，暖暖身子再工作！"

## 日期特殊祝福

### 工作日 (周一至周五)
- "新的一周/天，工作顺利！"
- "高效工作，快乐生活！"
- "目标明确，步步为营！"

### 周末 (周六、周日)
- "周末愉快，好好休息！"
- "工作之余，别忘了享受生活！"
- "放松心情，充电再出发！"

### 节假日
- "节日快乐！"
- "享受假期时光！"
- "休息是为了走更远的路！"

## 任务优先级模板

### 优先级1（最重要）
- "🔥 **今日核心**：[任务描述] - 今天必须完成！"

### 优先级2（重要）
- "⭐ **重点关注**：[任务描述] - 今天需要推进！"

### 优先级3（常规）
- "📝 **日常跟进**：[任务描述] - 按计划进行即可！"

## 完整示例

```
🌅 早安晗哥！皮休已上线 🐾

🌤️ **今日天气**：晴转多云
🌡️ **温度**：18°C ~ 26°C  
💨 **风力**：东南风2-3级
💧 **湿度**：65%
🎯 **空气质量**：良 (AQI 45)

🌟 **个性化祝福**：
阳光明媚，心情也会跟着灿烂起来！好天气带来好心情，今天一定顺利！

📋 **今日最重要事项**：
1. 🔥 **今日核心**：绩效证明文档完成 - 上午必须提交！
2. ⭐ **重点关注**：校区映射问题解决 - 今天截止！
3. 📝 **日常跟进**：SCRM问题跟进 - 至少推进2个问题

💡 **温馨提示**：
天气舒适，可以开窗通风，保持空气流通。记得多喝水，保持精力充沛！

新的一天，加油！✨
```
