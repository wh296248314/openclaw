# 2025年3月21日 记忆记录

## 重要事项记录

### 岗顶支持会议内容记录（2025年3月20日会议）
今天用户要求记录昨天岗顶支持会议的内容与待办事项。已创建详细会议记录文件。

**会议讨论的7个问题：**

1. **未跟进资源查询问题** - 已处理
   - 顾问查不到未跟进资源，但系统显示超过20条限制
   - 顾问表示已全部跟进

2. **退费数据重复导入问题** - 已解决
   - 不要重复导入退费数据
   - 已创建已退费公海人工转移机制

3. **已报读资源导入问题** - 已解决
   - 已报读资源是否可以不导入
   - 系统已有非在读与在读判定

4. **已毕业数据剔除问题** - 部分解决
   - 2025年已处理85%
   - 近一两个月很少反馈
   - 如遇到则人工转移到已毕业公海

5. **SCRM资源重复问题（二娃问题）** - 待处理
   - 以电话号码为判断标准
   - 家长有二胎时，资源仍给到第一个孩子
   - 名字、年级显示错误

6. **在读生显示未报名问题** - 待处理
   - 在读生显示未报名，需要跟进
   - 需要剔除在读生
   - 属于二娃问题

7. **在读生毕业后弟弟妹妹导入问题** - 待处理
   - 在读生毕业后，弟弟妹妹（小学）还会导进来
   - 需要辨别年级再导进系统
   - 属于二娃问题
