## 坑1：block_type 2 的字段名是 `text` 不是 `paragraph`

**发现时间**：2026-03-23
**发现小皮**：产品小皮
**严重程度**：⚠️ 中等（会导致API调用失败）

### 问题描述

在使用飞书文档API创建文本段落时，使用 `block_type: 2` 和 `paragraph` 字段名会报错：

```
错误码：1770001
错误信息：invalid param
```

### 原因

飞书文档API中 `block_type` 对应的字段名不是直观的名称：

| block_type | 字段名 | 说明 |
|------------|--------|------|
| 1 | page | 页面 |
| 2 | **text** | 文本段落（不是paragraph！） |
| 3 | heading1 | 一级标题 |
| 4 | heading2 | 二级标题 |
| 5 | heading3 | 三级标题 |
| 12 | bullet | 无序列表 |
| 13 | ordered | 有序列表 |

### 正确写法

```json
{
  "block_type": 2,
  "text": {
    "elements": [
      {
        "text_run": {
          "content": "这是文本内容",
          "text_element_style": {
            "bold": false,
            "inline_code": false,
            "italic": false,
            "strikethrough": false,
            "underline": false
          }
        }
      }
    ],
    "style": {
      "align": 1,
      "folded": false
    }
  }
}
```

### 错误写法

```json
{
  "block_type": 2,
  "paragraph": {  // ❌ 错误！应该用 text
    ...
  }
}
```
