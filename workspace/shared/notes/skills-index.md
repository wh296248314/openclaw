## 四、每日技能学习机制

### 执行时间
- 每天 9:30 前完成

### 检查流程
1. 检查 `shared/notes/` 是否有当日技能分享
2. 如未提交，@all 催促
3. 9:45 仍未提交，向皮休汇报

### 提交要求
- 路径：`shared/notes/技能分享-{小皮}-{YYYY-MM-DD}.md`
- 内容：技能名称、特点(3点)、应用场景、学习资源

## 五、技能创建流程

```
遇到问题 → 查找技能映射表
     ↓
是否有技能？
     │
    是 → 直接调用 → 解决 → 记录使用经验
     │
    否 → 评估：
     │     · 频率高 → 创建自建技能
     │     · 偶尔 → 记录为待建技能
     │     · 可组合 → 创建组合技能
     ↓
创建/组合技能 → 纳入映射表 → 同步给各小皮
```
