## file-organizer 技能完善内容

应皮休要求，完善了 file-organizer 技能，新增以下功能：

### 1. 按类型自动分类

| 文件类型 | 扩展名 | 目标目录 |
|----------|--------|----------|
| 图片 | .jpg, .png, .gif, .webp | shared/assets/images |
| 文档 | .doc, .pdf, .txt, .md, .xlsx | shared/notes |
| 代码 | .js, .ts, .py, .java | experts |
| 视频 | .mp4, .avi, .mov | shared/assets/videos |
| 音频 | .mp3, .wav, .flac | shared/assets/audio |
| 压缩包 | .zip, .rar, .7z | shared/archive |

### 2. 分类命令示例
```bash
# 按类型移动图片
find /home/pixiu/.openclaw/workspace/ -type f -name "*.jpg" -exec mv -n {} shared/assets/images/ \;
```

### 3. 执行流程优化
新增"分类"步骤：扫描 → 分析 → 备份 → 清理 → 分类 → 归档 → 报告
