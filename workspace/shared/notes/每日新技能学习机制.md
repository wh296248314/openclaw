# 每日新技能学习机制

> 制定者：小皮管家 | 日期：2026年3月18日 | 依据：皮休指示

---

## 机制说明

皮休要求各小皮每天早上9:30学习一个新技能，并准时汇报。

### 学习来源
- 官网资料
- 新知识
- 行业动态
- 工具推荐
- 技术文章

### 汇报时间
**每天早上9:30准时**

### 汇报对象
**皮休 + 所有人**

---

## 汇报模板

```markdown
# 新技能分享 - {小皮名称}

## 技能名称
xxx

## 技能特点
1. xxx
2. xxx
3. xxx

## 应用场景
xxx

## 学习资源
[链接]
```

## 汇报示例

```markdown
# 新技能分享 - 产品小皮

## 技能名称
Notion AI

## 技能特点
1. 智能文档生成
2. 自动总结归纳
3. 任务管理增强

## 应用场景
- 产品需求文档自动生成
- 会议记录智能整理

## 学习资源
https://notion.so
```

## 各小皮任务分配

| 小皮 | 重点关注领域 |
|------|-------------|
| 产品小皮 | 产品工具、AI应用、用户研究 |
| 设计小皮 | 设计工具、UI/UX趋势 |
| 研发小皮 | 开发框架、语言特性、开源工具 |
| 测试小皮 | 测试工具、质量保障 |
| 部署小皮 | DevOps、云计算、监控 |
| 审核小皮 | 规范标准、质量控制 |

- 每日汇报提交到：`shared/notes/新技能分享-{小皮名称}-{日期}.md`
- 统一汇总到：`shared/notes/每日新技能汇总-YYYY-MM.md`
