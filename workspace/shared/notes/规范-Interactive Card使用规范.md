# 规范-Interactive Card使用规范

> 2026-03-19 皮休发布

## 核心要求

**发送 Interactive Card 后必须做记录**，不能到时候让皮休回忆。

## 记录内容

每次发送 Interactive Card 后，记录：
- 发送时间
- 发送原因/目的
- Card 内容摘要
- 预期用户操作

## 记录位置

在当天的 memory 文件中记录，例如 `memory/2026-03-19.md`

## 示例

```markdown
## Interactive Card 发送记录

### 2026-03-19 XX:XX
- **用途**：xxx
- **内容**：xxx
- **预期**：xxx
```
