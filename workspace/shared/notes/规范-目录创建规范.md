# 目录创建规范

> 统一六小皮团队的目录标准 | 由小皮管家维护

---

## 一、必须目录

每个小皮workspace必须包含以下目录：

```bash
# 固定目录（必须存在）
memory/           # 日常记忆
skills/           # 技能链接
self-improving/   # 自我改进
```

## 二、目录层级

```
workspace/
├── memory/           # 记忆文件
│   └── YYYY-MM-DD.md
├── skills/           # 技能（软链接）
│   ├── file-organizer/
│   ├── mh-summarize/
│   └── ...
├── self-improving/   # 自我改进
│   ├── memory.md
│   ├── corrections.md
│   ├── index.md
│   ├── heartbeat-state.md
│   └── ...
├── docs/             # 工作文档（可选）
│   ├── 规范-xxx.md
│   └── 流程-xxx.md
└── knowledge/         # 知识库（可选）
    └── *.md
```

## 三、创建流程

### 1. 新增小皮时的目录初始化

```bash
# 1. 创建固定目录
mkdir -p memory/ skills/ self-improving/

# 2. 创建固定文件
touch MEMORY.md HEARTBEAT.md

# 3. 初始化 self-improving
mkdir -p self-improving/{projects,domains,archive}
touch self-improving/memory.md
touch self-improving/corrections.md
touch self-improving/index.md
touch self-improving/heartbeat-state.md

# 4. 链接核心技能
ln -s ../../skills/file-organizer skills/
ln -s ../../skills/mh-summarize skills/
ln -s ../../skills/planning-with-files skills/
ln -s ../../skills/self-improving skills/
ln -s ../../skills/find-skills skills/
ln -s ../../skills/using-superpowers skills/
```

### 2. 添加新文档

```bash
# 文档放在 docs/
# 使用标准命名：规范-话题-名称.md
touch docs/规范-需求管理.md
```

### 3. 添加知识库

```bash
# 知识库放在 knowledge/
# 文件命名：主题-内容.md
touch knowledge/产品基础知识.md
```

## 四、禁止事项

- ❌ 创建中文目录名
- ❌ 创建多层嵌套目录（超过3层）
- ❌ 在根目录放散落文件（除了固定配置文件）
- ❌ 用空格或特殊字符命名

## 五、检查脚本

```bash
# 检查目录是否符合规范
check_dirs() {
  for dir in memory skills self-improving; do
    if [ ! -d "$dir" ]; then
      echo "❌ Missing: $dir"
    else
      echo "✅ $dir exists"
    fi
  done
}
```
