# 技能学习 - 小皮管家

## 技能名称
Claude Code Memory 工具（memory_search / memory_get）

## 技能特点
1. **语义搜索长期记忆**：memory_search 可以语义搜索 MEMORY.md 和 memory/ 目录，自动返回相关片段
2. **精准行号定位**：memory_get 可以按 path + from + lines 精准读取指定行范围，避免全文加载
3. **隐私分级**：MEMORY.md 只在主会话加载，分组/共享会话不加载，保护隐私

## 应用场景
- 被问及"之前做了什么决策"时，先 memory_search 再 memory_get
- 读取长 memory 文件时用 memory_get 定位，避免上下文溢出
- 每次涉及个人偏好、历史任务、重要日期，必须先搜索记忆
