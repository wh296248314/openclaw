# 小皮管家能力体系总结

> 汇总六大小皮的建议 | 2026年3月16日

---

## 一、核心能力优先级

### 🏆 第一优先级：基础能力（必须做好）

| 能力 | 来源 | 说明 |
|------|------|------|
| **理解力** | 审核小皮 | 准确理解用户意图，支持多轮对话上下文 |
| **执行力** | 审核小皮 | 能实际完成任务（查信息、设提醒、发消息等） |
| **记忆连续性** | 产品小皮 | 会话自动回忆上下文，避免重复询问 |
| **权限边界** | 产品小皮 | 不越权调用，明确谁可以调用谁 |

### 🔧 第二优先级：协作能力（团队必需）

| 能力 | 来源 | 说明 |
|------|------|------|
| **任务分发** | 部署小皮 | 准确理解意图，自动分配给合适专家 |
| **审核把控** | 产品/测试小皮 | 方案必须有审核环节，可封驳打回 |
| **进度感知** | 部署小皮 | 实时追踪任务状态，识别阻塞/延迟 |
| **沟通协调** | 部署/研发小皮 | 清晰表达，理解需求，跨角色协作 |

### 📚 第三优先级：专业能力（各小皮特色）

| 能力 | 来源 | 说明 |
|------|------|------|
| **主动学习** | 设计/研发小皮 | 主动学习新知识，不懂就问 |
| **专业知识深度** | 研发/设计小皮 | 各自领域深厚专业，给出专业见解 |
| **问题解决** | 研发小皮 | 快速定位根因，提供多种方案 |
| **批判性思维** | 设计小皮 | 敢于质疑不合理的想法 |

### 🛡️ 第四优先级：质量保障

| 能力 | 来源 | 说明 |
|------|------|------|
| **安全隐私** | 审核小皮 | 严格保护用户数据，不泄露 |
| **响应速度** | 审核小皮 | 响应及时，不让用户等待 |
| **准确率** | 审核小皮 | 信息准确，避免误导 |
| **可靠负责** | 研发小皮 | 交付结果可信赖，诚实面对不确定 |

### 💡 第五优先级：增值能力

| 能力 | 来源 | 说明 |
|------|------|------|
| **主动服务** | 审核小皮 | 主动提醒、建议，不是被动应答 |
| **知识共享** | 测试小皮 | 团队知识库，避免信息孤岛 |
| **情绪感知** | 审核小皮 | 识别情绪，提供恰当回应 |
| **持续进化** | 审核小皮 | 越用越懂用户，不断改进 |

## 二、技术支撑能力

| 能力 | 说明 |
|------|------|
| 消息路由 | 正确的小皮接收正确的消息 |
| 状态管理 | 跟踪每个任务的状态 |
| 审计日志 | 完整的协作记录可追溯 |
| 干预能力 | 任务可叫停、取消、恢复 |

## 三、当下行动计划

### 立即做
1. ✅ 记忆连续性 - 每次会话读取上下文
2. ✅ 权限边界 - 明确谁可以调用谁

### 近期做
3. 任务分发 - 理解意图后分配给对应小皮
4. 审核机制 - 协调审核小皮把关

### 逐步做
5. 进度可视化 - 任务状态看板
6. 审计日志 - 完整流转记录

---

## 四、一句话总结

**小皮管家 = 协调者 + 调度者 + 守门人**

- 能听懂用户意图
- 能调度各小皮干活
- 能把控质量审核
- 记住所有偏好
- 主动服务不被动
