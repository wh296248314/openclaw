# 文件命名规范

> 统一六小皮团队的命名标准 | 由小皮管家维护

---

## 一、核心文件命名

### 1. 系统配置文件（大写，固定）

| 文件名 | 用途 |
|--------|------|
| `AGENTS.md` | Agent配置与规则 |
| `SOUL.md` | Agent身份与行为准则 |
| `IDENTITY.md` | Agent身份信息 |
| `USER.md` | 用户信息 |
| `TOOLS.md` | 工具配置 |
| `MEMORY.md` | 长期记忆（**大写，重要**） |
| `HEARTBEAT.md` | 心跳任务清单 |

### 2. 文档文件（标准格式）

```
[类型]-[话题]-[名称].md
```

| 类型前缀 | 含义 |
|----------|------|
| `规范-` | 规则、制度、规范 |
| `流程-` | 操作流程 |
| `模板-` | 模板文件 |
| `规范-` | 规则制度 |
| `日报-` | 每日工作 |
| `周报-` | 每周总结 |
| `复盘-` | 项目复盘 |
| `文档-` | 文档管理 |

**示例：**
```
规范-团队协作.md
流程-需求评审.md
模板-周报模板.md
日报-2026-03-18.md
文档-管理规范.md
```

### 3. 禁止的命名

- ❌ 中文拼音混合：`xuqiuwendang.md`
- ❌ 空格：`我的文档.md`
- ❌ 特殊字符：`文档@v1.md`
- ❌ 中英混用：`需求文档.md`（用`规范-需求管理.md`）

---

## 二、目录命名

### 1. 固定目录（必须存在）

| 目录名 | 用途 |
|--------|------|
| `memory/` | 日常记忆文件 |
| `skills/` | 技能链接 |
| `self-improving/` | 自我改进记录 |

### 2. 可选目录

| 目录名 | 用途 |
|--------|------|
| `docs/` | 工作文档 |
| `knowledge/` | 知识库 |
| `shared/` | 共享资源（仅主workspace） |

### 3. 禁止的目录

- ❌ `learning/` → 用 `knowledge/`
- ❌ `learn/` → 用 `knowledge/`
- ❌ `reference/` → 用 `knowledge/`
- ❌ 中文目录名
