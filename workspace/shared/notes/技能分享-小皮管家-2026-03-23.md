# 技能分享 - 小皮管家 - 2026-03-23

## 技能名称
**任务规划与进度追踪（Planning with Files）**

## 技能特点

1. **Manus风格文件规划**
   - 创建 `task_plan.md`、`findings.md`、`progress.md` 三文件体系
   - 结构化记录任务分解、发现、进展
   - 支持自动会话恢复，断点续传

2. **进度可视化追踪**
   - 使用 `## 进度` 章节记录各阶段完成状态
   - 符号化标记（✅ 已完成 / 🔄 进行中 / ⏳ 待启动）
   - 支持多任务并行追踪

3. **上下文管理**
   - 每个任务独立文件，避免上下文污染
   - 支持 `/clear` 后自动恢复会话
   - 适合复杂、长周期项目协调

## 应用场景

- **管家协调多小皮任务**：并行追踪6个专家小皮的工作进展
- **长周期项目**：SCRM竞品分析等多日任务的分阶段管理
- **会议纪要**：朝会/讨论的结论记录与执行追踪
- **问题排查**：系统故障的发现、分析、修复进度记录
