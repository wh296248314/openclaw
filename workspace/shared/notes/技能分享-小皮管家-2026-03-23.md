# 技能分享 - 小皮管家

## 技能名称
**PM2 进程管理器** - Node.js应用生产环境部署神器

---

## 技能特点

### 1. 进程守护，自动重启
- 应用崩溃后自动重启，保持服务持续运行
- 支持负载均衡（cluster模式），充分利用多核CPU
- 重启次数限制，避免无限崩溃循环

### 2. 零停机部署
- 支持 reload 命令，优雅重启不丢请求
- 代码更新后热加载，用户无感知
- 延迟重启，等待连接处理完毕

### 3. 资源监控与日志管理
- 实时监控CPU/内存使用率
- 统一日志管理，自动切割日志文件
- 支持日志归档和查询

---

## 应用场景

### 场景1：部署OpenClaw Gateway
```bash
pm2 start openclaw-gateway --name openclaw
pm2 save  # 保存进程列表
pm2 startup  # 设置开机自启
```

### 场景2：部署edict Dashboard
```bash
pm2 serve /path/to/dist --name edict --port 7891
pm2 list  # 查看状态
pm2 logs edict  # 查看日志
```

### 场景3：Node.js API服务
```javascript
// ecosystem.config.js
module.exports = {
  apps: [{
    name: 'xiaopi-api',
    script: './server.js',
    instances: 'max',
    exec_mode: 'cluster',
    env: { NODE_ENV: 'production' }
  }]
};
```
