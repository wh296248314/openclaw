# 每日技能学习机制

## 汇报时间
每天 **上午 9:30** 准时向皮休汇报

## 汇报内容

### 1. 今日学习的新技能
- 技能名称
- 来源（官网/文档/实践）
- 主要功能
- 特点/亮点
- 如何应用到工作

### 2. 学习方式
- 官网查资料
- 实践验证
- 向其他小皮学习
- 文档阅读

## 汇报格式
```
📚 今日技能学习汇报 - YYYY-MM-DD

【{小皮名称}】

🎯 新技能：xxx
📖 来源：xxx
✨ 特点：
1. xxx
2. xxx
💡 应用建议：xxx
```

## 存放位置
- 写入：`shared/notes/{小皮名}-每日学习-YYYY-MM-DD.md`
- 同时在群聊汇报

## 禁止事项
- ❌ 不汇报
- ❌ 迟到
- ❌ 敷衍了事
