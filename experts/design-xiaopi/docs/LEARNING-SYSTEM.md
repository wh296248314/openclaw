# 设计学习体系

## 1. 定时任务配置

### 学习任务定时任务

```bash
# 每日设计趋势学习（每天早上9点）
openclaw cron add \
  --name "每日设计趋势学习" \
  --agent design-xiaopi \
  --cron "0 9 * * *" \
  --tz "Asia/Shanghai" \
  --message "执行每日设计趋势学习任务：搜索最新的UI设计趋势、交互设计案例、用户体验研究" \
  --announce

# 每周设计灵感收集（每周一早上10点）
openclaw cron add \
  --name "每周设计灵感收集" \
  --agent design-xiaopi \
  --cron "0 10 * * 1" \
  --tz "Asia/Shanghai" \
  --message "执行每周设计灵感收集：搜索本周优秀设计案例、创意设计趋势、设计文章" \
  --announce

# 每月设计趋势深度分析（每月1号早上10点）
openclaw cron add \
  --name "每月设计趋势深度分析" \
  --agent design-xiaopi \
  --cron "0 10 1 * *" \
  --tz "Asia/Shanghai" \
  --message "执行本月设计趋势深度分析：分析当月设计趋势变化、热门设计模式、用户研究洞察" \
  --announce
```

## 2. 关注领域配置

### 设计领域分类

#### 核心领域
- **UI设计趋势**：界面设计最新趋势、配色方案、排版设计
- **交互设计**：交互模式、动效设计、用户体验最佳实践
- **用户体验**：用户研究方法、可用性测试、用户旅程地图
- **设计系统**：组件库设计、设计规范、品牌设计语言

#### 细分领域
- **移动端设计**：APP设计、小程序设计、移动端交互
- **响应式设计**：多端适配、响应式布局
- **数据可视化**：图表设计、数据仪表盘、信息图
- **无障碍设计**：无障碍访问、包容性设计

#### 新兴领域
- **AI设计**：AI辅助设计工具、智能交互界面
- **空间设计**：AR/VR界面设计、3D交互
- **语音设计**：语音交互设计、VUI设计

## 3. 学习工作流

### 日常学习流程
1. **信息获取**
   - 搜索最新设计文章和案例
   - 抓取设计网站内容（Dribbble、Behance、UI中国）
   - 关注设计Newsletter和博客

2. **信息筛选**
   - 过滤与当前项目相关的内容
   - 识别高价值学习资源
   - 标记需要深入研究的主题

3. **信息整理**
   - 记录关键设计趋势
   - 保存优秀设计案例
   - 更新设计灵感库

4. **知识应用**
   - 将学习成果应用到当前项目
   - 与团队分享有价值的内容
   - 更新设计规范和组件库

### 每周学习重点
- 周一：本周设计趋势概览
- 周二：优秀案例深度分析
- 周三：交互设计模式研究
- 周四：用户体验研究分享
- 周五：设计工具和技巧学习

### 每月学习主题
- 第一周：设计趋势深度分析
- 第二周：设计系统更新和完善
- 第三周：用户研究和方法论
- 第四周：设计工具和技术更新

## 5. 定时任务管理

### 查看定时任务
```bash
openclaw cron list
```

### 运行定时任务（测试）
```bash
openclaw cron run "每日设计趋势学习"
```

### 删除定时任务
```bash
openclaw cron rm "每日设计趋势学习"
```
