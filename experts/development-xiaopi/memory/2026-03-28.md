## 小皮OS 黑屏问题修复

### 问题描述
小皮OS页面大部分Tab（朝会、官员、技能、旨库等）显示黑屏

### 根因分析
1. **CSS变量缺失**：`--accent` 未定义，导致依赖它的样式无法正确渲染
2. **CSS display规则问题**：`.page { display: none }` 导致页面默认隐藏
3. **WebSocket服务问题**：只监听localhost，远程无法连接
4. **CORS配置问题**：HTTP服务未配置跨域
5. **News.jsx缺少useEffect导入**：导致要闻页面JS错误
6. **Archive.jsx语法错误**：`selectedArchive批复` 无效应为 `selectedArchive?.批复`

### 修复内容
1. ✅ 修复CSS变量缺失 - 添加 `--accent: #6a9eff`
2. ✅ **修复CSS display规则** - `.page { display: none }` → `{ display: block }` （主要根因）
3. ✅ WebSocket改为监听 `0.0.0.0:1212`
4. ✅ HTTP服务添加 `--cors "*"`
5. ✅ 前端WebSocket URL改为动态获取 `ws://${window.location.hostname}:1212`
6. ✅ News.jsx 添加 useEffect 导入
7. ✅ Archive.jsx 修复 `selectedArchive批复` → `selectedArchive?.批复`
