## 一、尘锋SCRM

### 基本信息
| 项目 | 内容 |
|------|------|
| 定位 | 企业微信SCRM头部厂商 |
| 成立 | 2019年 |
| 融资 | 多轮融资，B端头部 |
| 官网 | chatifen.com |

### 核心功能
- **营销获客**：裂变海报、智能名片、渠道活码
- **客户管理**：企业微信联系人同步、客户画像、客户生命周期管理
- **销售管理**：SOP自动化、跟进提醒、话术库
- **社群运营**：群发助手、群积分、社群SOP
- **数据分析**：流量分析、转化漏斗、员工绩效报表
- **开放能力**：API/SDK支持，可对接ERP/CRM

### 目标客群
中大型企业为主，覆盖金融、教育、零售、医疗等

### 定价策略
按坐席/年收费，标准版/专业版/旗舰版，价格区间约 **300~800元/坐席/年**

### 优势
- 企业微信原生集成深，客户量积累大
- 功能模块全，SOP自动化成熟
- 客服响应快，服务体系完善

### 劣势
- 价格偏高，中小企业门槛高
- 部分功能依赖企业微信版本更新
- UI/体验相对传统

## 二、微盛SCRM

### 基本信息
| 项目 | 内容 |
|------|------|
| 定位 | 企业微信生态SCRM，增长最快 |
| 成立 | 2015年（微盛网络科技） |
| 融资 | 腾讯领投C轮，数亿元 |
| 官网 | weishengkeji.com |

### 核心功能
- **获客工具**：裂变获客、渠道码、抽奖活动
- **运营管理**：群发助手、自动拉群、标签管理
- **销售跟进**：跟进记录、客户阶段管理、会话存档
- **数据分析**：ROI统计、员工行为分析、转化分析
- **企业微信插件**：群红包、话术推荐、会话监管
- **多平台支持**：支持抖音、快手等多渠道接入

### 目标客群
中腰部企业为主，零售、电商、教育、金融均有覆盖

### 定价策略
按员工账号收费，约 **200~600元/账号/年**，有免费试用期

### 优势
- 增长势头猛，腾讯官方投资背书
- 多平台打通能力强（企业微信+抖音+快手）
- 操作简单，上手快，中小企业友好
- 生态合作广泛

### 劣势
- 功能深度相对头部厂商稍浅
- 大型企业定制化能力有限
- 价格战压力大

## 三、探马SCRM

### 基本信息
| 项目 | 内容 |
|------|------|
| 定位 | 销售导向SCRM，主打销售管理 |
| 成立 | 2019年 |
| 融资 | 多轮融资，B轮左右 |
| 官网 | tanma.cn |

### 核心功能
- **销售管理**：线索池管理、销售漏斗、商机跟踪、业绩目标
- **CRM基础**：客户管理、联系人、跟进记录
- **企业微信集成**：聊天存档、会话质检、敏感词监控
- **营销工具**：内容物料库、朋友圈营销
- **数据驾驶舱**：销售排名、转化率、客户质量分析
- **移动端**：销售外勤打卡、拜访记录

### 目标客群
以销售团队为核心需求的B2B企业、金融、制造业、教育

### 定价策略
按人数收费，约 **300~700元/人/年**

### 优势
- 销售管理功能深，CRM底子扎实
- 数据驾驶舱直观，管理层友好
- 移动端外勤管理能力强

### 劣势
- 营销获客模块相对弱
- 社群运营能力不如尘锋/微盛
- 生态整合能力有待提升

- 定制化能力弱
- 无大客户案例背书，品牌信任度低

## 五、腾讯企点

### 基本信息
| 项目 | 内容 |
|------|------|
| 定位 | 腾讯官方，企业微信/QQ/电话多渠道客服+CRM |
| 归属 | 腾讯云&腾讯企业产品线 |
| 成立 | 2016年（企点前身） |
| 官网 | qidian.qq.com |

### 核心功能
- **多渠道客服**：企业微信、QQ、微信公众号、电话热线统一接入
- **CRM基础**：客户信息管理、客户分组、标签
- **营销触达**：短信、邮件、微信模板消息群发
- **数据分析**：接待统计、会话分析、客户画像
- **企点客服机器人**：智能客服，自动问答
- **与腾讯生态深度绑定**：企业微信、腾讯会议、腾讯云无缝衔接

### 目标客群
全规模企业，尤其已使用腾讯系产品（企业微信/腾讯云）的企业

### 定价策略
按坐席+用量混合收费，约 **300~1000元/坐席/年**，客服机器人为增值付费

### 优势
- 腾讯官方品牌，信任度最高，数据安全性强
- 与企业微信、腾讯会议、腾讯云等深度集成
- 多渠道（微信+QQ+电话）统一管理能力强
- 持续迭代，腾讯生态资源加持

### 劣势
- SCRM营销模块相对弱，更偏客服/接待
- 销售管理（SOP、漏斗）不如专业SCRM厂商
- 定制化能力受限于腾讯产品路线
- 商务流程相对传统，大客户需对接销售

## 七、关键洞察

### 1. 市场格局
- **头部**：尘锋、微盛占据企业微信SCRM主要市场，差异化明显
- **腰部**：探马以销售管理切入细分市场
- **长尾**：蝉客主打低价轻量，争夺小微企业
- **特殊位**：腾讯企点依托官方身份占据企业微信配套市场

### 2. 竞争核心
- 企业微信生态深度集成是入场券
- 营销自动化+社群运营是差异化关键
- 销售管理（CRM底子）是B2B场景核心需求

### 3. 威胁与机会
- **腾讯官方**：企点持续迭代可能挤压第三方SCRM空间
- **企点弱点**：营销模块弱，专业SCRM厂商仍有生存空间
- **中小企业**：蝉客等低价竞品会持续价格战
- **垂直行业**：婚摄、医疗、教育等垂直SCRM有机会
