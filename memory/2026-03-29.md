## Git sync bug 修复（15:22）

**问题根因**：`git add -A` 对中文文件名会静默失败，改为 `git add -u`（只加已跟踪文件的修改）

**正确用法**：
```bash
git add -u    # ✅ 只加已跟踪文件的修改，中文名不报错
git add -A    # ❌ 对中文文件名会静默失败
```
