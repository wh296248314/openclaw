#### 2. 数据时间字段定义梳理

**字段定义标准化：**

| 字段名称 | 定义 | 说明 | 相同点/不同点 |
|----------|------|------|---------------|
| **导入时间** | 表格导入、系统对接、表单导入时，首次创建资源/客户的时间 | 系统自动记录 | 与创建时间值相同，但场景不同 |
| **创建时间** | 人工录入时，创建资源/客户的时间 | 用户手动创建 | 与导入时间值相同，但场景不同 |
| **参与活动时间** | 第2-n次参与活动时记录的时间 | 需要系统新增字段 | 导入时间 = 首次参与活动时间 |

**重要说明：**
- 导入时间和创建时间在数值上相同，但应用场景不同
- 导入时间和首次参与活动时间相同

#### 4. 公海时间字段关系梳理

**字段关系分析：**

| 字段名称 | 定义 | 当前问题 | 优化需求 |
|----------|------|----------|----------|
| **分配时间** | 资源分配给顾问的时间 | - | - |
| **领取时间** | 顾问从公海领取资源的时间 | 公海没有此字段 | 需要明确字段定义 |
| **转移时间** | 资源在公海间转移的时间 | - | - |
| **离开公海时间** | 不管什么方式出公海的时间（包含领取、分配、转移） | 在"公海非在读已分配"页面中，离开公海时间没有包括领取时间 | 需要修复逻辑 |

**当前问题总结：**
1. 公海缺少"领取时间"字段定义
2. 离开公海时间应包含所有出公海的方式（领取、分配、转移）
3. 在"公海非在读已分配"页面中，离开公海时间的计算逻辑不完整
