# 早安问候智能模板

## 动态变量
问候语会自动包含以下信息：
1. **当前日期**：{{date:YYYY年MM月DD日}} {{date:dddd}}
2. **地点**：广州天河区
3. **时间**：{{date:HH:mm}}
4. **是否周末**：{{#if date:isWeekend}}是{{else}}否{{/if}}

## 智能祝福逻辑

### 基于星期的祝福
- **周一**：新的一周开始啦！充满能量，高效工作！
- **周二**：周二加油！工作渐入佳境！
- **周三**：周中啦！坚持就是胜利！
- **周四**：周四了，周末就在眼前！
- **周五**：周五快乐！完美收官这一周！
- **周六**：周末愉快！好好放松休息！
- **周日**：周日安好！为新的一周做准备！

### 基于时间的祝福
- **早上 (5:00-9:00)**：清晨好！一日之计在于晨！
- **上午 (9:00-12:00)**：上午好！高效工作时段！
- **中午 (12:00-14:00)**：午安！记得适当休息！
- **下午 (14:00-18:00)**：下午好！保持专注！
- **晚上 (18:00-22:00)**：晚上好！放松一下！
- **深夜 (22:00-5:00)**：夜深了，注意休息！

## 任务提取逻辑
问候语会自动从 `memory/{{date:YYYY-MM-DD}}.md` 中提取：
1. 寻找"## 今日重要任务"或"### 1."部分
2. 提取前3个任务点
3. 按优先级标记：🔥今日核心、⭐重点关注、📝日常跟进

## 天气提醒占位
虽然无法实时获取天气，但可以提醒：
"记得查看今日天气，合理安排出行和着装！"

## 完整模板

```
🌅 早安晗哥！皮休已上线 🐾

📅 **今天是**：{{date:YYYY年MM月DD日}} {{date:dddd}}
📍 **地点**：广州天河区
⏰ **当前时间**：{{date:HH:mm}}

🌟 **个性化祝福**：
{{#if date:isWeekend}}周末愉快！好好休息，充电再出发！{{else}}{{#switch date:dddd}}
{{#case "星期一"}}新的一周开始啦！充满能量，高效工作！{{/case}}
{{#case "星期二"}}周二加油！工作渐入佳境！{{/case}}
{{#case "星期三"}}周中啦！坚持就是胜利！{{/case}}
{{#case "星期四"}}周四了，周末就在眼前！{{/case}}
{{#case "星期五"}}周五快乐！完美收官这一周！{{/case}}
{{#default}}新的一天开始啦！高效工作，快乐生活！{{/default}}
{{/switch}}{{/if}}

📋 **今日最重要事项**（从 memory/{{date:YYYY-MM-DD}}.md 读取）：
1. 🔥 **今日核心**：[自动提取任务1或默认]
2. ⭐ **重点关注**：[自动提取任务2或默认]  
3. 📝 **日常跟进**：[自动提取任务3或默认]

🌤️ **天气提醒**：
记得查看今日天气，合理安排出行和着装！广州天气多变，记得关注实时预报。

💡 **温馨提示**：
{{#if date:isWeekend}}周末记得放松，但也别忘了规划下周工作！{{else}}记得先查看今日任务清单，合理安排时间。遇到复杂任务记得分步处理哦！{{/if}}

新的一天，加油！✨
```

## 默认任务（当无法读取文件时）
1. 🔥 **今日核心**：处理优先级最高的任务
2. ⭐ **重点关注**：推进重要项目进展  
3. 📝 **日常跟进**：更新工作记录和沟通

## 特殊日期提醒
- **月初 (1-3号)**：记得制定本月计划！
- **月末 (27-31号)**：记得进行月度总结！
- **节假日**：节日快乐！享受假期时光！
- **生日/纪念日**：特殊的日子，特别的祝福！
